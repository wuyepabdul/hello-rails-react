# Hello React Rails

> This project is an implementation of connect react-rails gem. Built with Ruby, Javascript and Ruby on Rails

## Built With

- Ruby
- Ruby on Rails
- JavaScript
- CSS

## Getting Started

```
git clone https://github.com/wuyepabdul/hello-rails-react

cd hello-rails-react
rails db:create
rails db:migrate
rails db:seed

rails server


```

## Tests

```
git clone https://github.com/wuyepabdul/hello-rails-react
cd hello-rails-react
bundle install
gem install rspec
rspec
```
